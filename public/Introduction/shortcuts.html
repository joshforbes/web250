Shortcut Description
Cmd + 1 Toggles project sidebar
Cmd + Option + L Format the current file to according to style guide
Control + Space Opens the code completion popup
Shift + Shift (double tap) Opens a 'go to anything' window, which allows you to search through everything in your project or through PHPStorm config files
Option + Up Arrow Select increasing code blocks. Example: one tap selects current word, two taps selects everything within current tags, three taps selects next level up, etc..
Cmd + F Find
Control + D Debug
Cmd + O Go to class
Cmd + P Go to file
Cmd + L Go to line
Cmd + E Recent Files Popup: shows recently opened files to allow for quick switching between
Cmd + U Go to Super Class/Super Method
Control + Up / Control + Down Jump to Next / Previous Method
Cmd + D Enables Multiple Cursor (this is a custom binding I have setup) (mimics the behavior of sublime texts multiple cursor)
Cmd + , Opens settings window
Cmd + W Close active editor tab
Cmd + / Comments out current selection with line comment
Cmd + Option + / Comments out current selection with block comment
Cmd + Up Opens the Navigation menu - allows for quick movement through the directory structure
Cmd + N Opens the New context menu in the current folder with using the navigation menu. Allows for quick creation of files and directories
